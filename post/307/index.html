Search executable file with find command

发表于2008-06-16 16:24:15

今天在折腾程序，突然目录结构变了，所以想着把所有的编译生成的文件给删了，然后把源代码扔到另一个目录里面再去编译，一个一个手动删太累了，于是懒人我就想到了fi nd命令。

man了一下，找到了一个 -perm 选项，在后面可以像 chmod 一样指定421的权限位，于是乎就简单了，一条命令搞定：

find . -perm /u+x -type f -exec -rm {} \;

千万记得在exec前要打印出来看看撒…….
