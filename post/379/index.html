Linux下的十六进制编辑器

发表于2009-01-03 20:45:48

关键字： Linux, hex editor, cli

折腾东西，需要找一个命令行下的十六进制编辑器，最后在这篇文章中找到了办法。其实就是在用xxd看了，编辑完后，用xxd -r命令把它给恢复回去，然后再写回文件就行了，嗯。

vim file
:%!xxd
...
edit
...
:%!xxd -r
